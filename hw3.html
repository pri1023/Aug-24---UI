HW3 - CSS SELECTORS
There are many selectors

The following is a list of selectors.

  1. * - targets all the elements on the page ||
  2. #X - used for a class ||
  3. .X - used for an id ||
  4. X,Y - applies the properties to two different elements||
    • X - applies the properties to one element||
    • X.link - applied to those anchors which are not yet clicked on ||
    • X.visited - applied to those anchors which have been clicked on ||
    • X+Y - applied only if X is next to Y
    • X~Y - applied to all Y elements that follow X
    • X>Y - applied only if Y is a child of X ||
    • X[href="#"] - apply properties to those links listed here ||
    • X[href*=""] - star represents all those links that match the keyword ||
      • X[href$=".jpg"] - $ refers to the end of the string, so this is applied to all those with .jpg extensions ||
      • X[data-*="foo"] - this is used to target more than one images file extension type.
      • X[foo~="bar"] - This is applied to all those elements which contain the exact attribute value pairs
        1. X:checked - Targets only those values which eqate to checked
        2. X:hover - applied when the cursor hovers over the link or text
        3. X:not(selector) - selects all elements of on type except the one specified
        4. X[attribute=value] - selects all elements with attribute value
        5. X:first-child - applies properties to the first child
        6. X:nth-child(n) - applies properties to the nth child
